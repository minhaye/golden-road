Mô tả:

Thêm test suite cơ bản, script build, và thiết lập CI (GitHub Actions) để chạy build/test tự động.

Checklist:
- [ ] Thêm unit tests cho logic quan trọng
- [ ] Viết script build/run tests
- [ ] Thiết lập `.github/workflows/ci.yml` đơn giản
- [ ] Kiểm thử workflow trên PR
