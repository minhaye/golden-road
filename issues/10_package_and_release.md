Mô tả:

Chuẩn hóa quá trình build và đóng gói thành file JAR có thể chạy độc lập.

Checklist:
- [ ] Viết script build (ant/gradle/maven hoặc javac+jar)
- [ ] Đảm bảo assets được đóng gói trong JAR hoặc đặt bên ngoài và chỉ định path
- [ ] Tạo hướng dẫn chạy trong README
- [ ] Tạo release tag khi sẵn sàng
